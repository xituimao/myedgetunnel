# 快速部署指南

## 服务器信息
- **IP地址**: YOUR_SERVER_IP
- **用户名**: root

## 快速部署（推荐）

### 方法一：自动部署脚本

如果你的电脑上有 `sshpass` 工具：

```bash
chmod +x deploy.sh
./deploy.sh
```

### 方法二：手动部署（分步指导）

```bash
chmod +x deploy-manual.sh
./deploy-manual.sh
```

这个脚本会一步步引导你完成部署。

### 方法三：完全手动部署

1. **登录服务器**
```bash
ssh root@YOUR_SERVER_IP
```

2. **安装 Node.js**
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt-get install -y nodejs
```

3. **安装 PM2**
```bash
npm install -g pm2
```

4. **在本地打包项目**（在另一个终端）
```bash
tar -czf myedgetunnel.tar.gz package.json server.js _worker.js ecosystem.config.cjs .env.example
scp myedgetunnel.tar.gz root@YOUR_SERVER_IP:/opt/
```

5. **在服务器上部署**
```bash
cd /opt
mkdir -p myedgetunnel
cd myedgetunnel
tar -xzf ../myedgetunnel.tar.gz

# 安装依赖
npm install --production

# 创建配置文件
cp .env.example .env

# 启动服务
pm2 start ecosystem.config.cjs
pm2 save
pm2 startup

# 配置防火墙
ufw allow 8080/tcp
```

## 访问应用

部署完成后：

- **主页**: http://YOUR_SERVER_IP:8080
- **管理后台**: http://YOUR_SERVER_IP:8080/admin
- **管理密码**: 通过 ADMIN 环境变量设置

⚠️ **重要**：首次登录后请立即修改管理员密码！

## 常用命令

```bash
# 查看状态
pm2 status

# 查看日志
pm2 logs myedgetunnel

# 重启应用
pm2 restart myedgetunnel

# 停止应用
pm2 stop myedgetunnel
```

## 修改配置

编辑配置文件：
```bash
ssh root@YOUR_SERVER_IP
nano /opt/myedgetunnel/.env
```

修改后重启：
```bash
pm2 restart myedgetunnel
```

## 遇到问题？

1. 查看日志：`pm2 logs myedgetunnel --lines 100`
2. 检查端口：`netstat -tunlp | grep 8080`
3. 检查防火墙：`ufw status`
4. 重启应用：`pm2 restart myedgetunnel`
