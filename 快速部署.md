## 快速部署（推荐）

### 方法一：自动部署脚本

如果你的电脑上有 `sshpass` 工具：

```bash
chmod +x deploy.sh
./deploy.sh
```

### 方法二：手动部署（分步指导）

```bash
chmod +x deploy-manual.sh
./deploy-manual.sh
```

这个脚本会一步步引导你完成部署。

### 方法三：完全手动部署

1. **登录服务器**
```bash
ssh root@69.5.7.220
# 输入密码: Sx@3964117
```

2. **安装 Node.js**
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt-get install -y nodejs
```

3. **安装 PM2**
```bash
npm install -g pm2
```

4. **在本地打包项目**（在另一个终端）
```bash
tar -czf myedgetunnel.tar.gz package.json server.js _worker.js ecosystem.config.cjs .env.example
scp myedgetunnel.tar.gz root@69.5.7.220:/opt/
```

5. **在服务器上部署**
```bash
cd /opt
mkdir -p myedgetunnel
cd myedgetunnel
tar -xzf ../myedgetunnel.tar.gz

# 安装依赖
npm install --production

# 创建配置文件
cp .env.example .env

# 启动服务
pm2 start ecosystem.config.cjs
pm2 save
pm2 startup

# 配置防火墙
ufw allow 8080/tcp
```

